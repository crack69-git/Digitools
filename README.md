Project Name: digitools247 
It is a digital React based web application that is used for purchasing premium subscription of different types of apps.

**Technologies Used**
- React
- JavaScript
- Tailwind CSS
- Daisy UI
- HTML
- CSS
- React toastify

**Key Features**
- The number of Product Available in the cart right now is shown in Cart and in navbar cart icon and total amount is displayed in the cart total section.
- Removing product from the cart will decrese the total amount and also the number of product shown will also decrease.
- Used React toasify fo interactive notification while adding to cart,removing from cart and checkout from cart.
